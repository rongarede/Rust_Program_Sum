# 猜数字游戏 (Guessing Game)

一个用 Rust 编写的简单猜数字游戏。

## 游戏规则

- 程序会随机生成一个 1 到 100 之间的数字
- 玩家需要猜测这个数字
- 程序会提示猜的数字是太大还是太小
- 继续猜测直到猜对为止

## 运行游戏

确保已安装 Rust，然后在项目目录下运行：

```bash
cargo run
```

## 依赖

- `rand = "0.8"` - 用于生成随机数

## 项目结构

```
guessing_game/
├── Cargo.toml
├── README.md
└── src/
    └── main.rs
```

## 功能特性

- 随机数生成
- 用户输入验证
- 错误处理
- 中文界面提示
- 循环游戏直到猜对
